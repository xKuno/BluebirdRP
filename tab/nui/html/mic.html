Microphone Setup

Use of a Microphone is required by Server Rules

You must have a microphone or you will be removed from the server. Here is how to set it up if you're unsure.

You can find microphone settings in the FiveM Pause Menu (see image below)

Ensure that that your microphone is plugged in!

Press Escape or Left Ctrl + P to enter pause menu

1. Go to Settings

2. Choose Voice Chat

3. Toggle Voice chat off and back on

4. Turn up the volume so you can hear, also ensure you have the correct output device selected (eg: headphones, speakers etc.)

5. Turn up the microphone volumes so others can hear you, also ensure you have the correct input device selected.
